FINEST MWAMBA WEBSITE - HOW TO EDIT

1. Open index.html in a browser to preview the site.
2. To replace photos, add images to assets/images/ using these suggested names:
   - author-finest.jpg
   - student-life-zambia-cover.jpg
   - second-book-cover.jpg
   - event-1.jpg
   - event-2.jpg
   - event-3.jpg
   - event-4.jpg
   - event-5.jpg
   - event-6.jpg

3. Then replace the placeholder blocks in index.html with image tags, for example:
   <img src="assets/images/author-finest.jpg" alt="Finest Mwamba" />

4. Update contact details in the Contact section:
   - Email
   - WhatsApp
   - Facebook / Instagram / TikTok / LinkedIn links

5. Update Events Done and Upcoming Events with real dates, venues and links.

6. To upload online:
   - Netlify: drag and drop this whole folder or the ZIP file.
   - GitHub Pages: upload all files to a repository and enable Pages.
